# IITA / ABCP / PHMD annual-report finds (≈1989–1995)

Search notes for biologicalcontrol.org folklore / alumni archive.  
Searched: CGSpace, FAO Open Knowledge, IITA biblio URLs, USAID/World Bank catalogs, Google Scholar-style web hits, Google Books metadata. Firecrawl CLI was not available in this environment (`firecrawl` / `npx firecrawl` failed); CGSpace REST API + WebSearch/WebFetch used instead.

**Bottom line:** The open digital trail for this period is strongest in **IITA institutional annual reports** (biocontrol folded into “Biological Control” then “Plant Health Management” chapters) and the **1995 Fourth External Programme & Management Review**. A true standalone **PHMD / Africa-wide BCP annual-report series for 1989–1995 is largely missing online**; the earliest clearly titled PHMD annual reports on CGSpace are **1998+**. One 1994 item is explicitly cited as a *Plant Health Management Division Annual Report* (EScaPP) but has **no PDF on CGSpace** (Google Books only).

## D. Dead ends (brief)

- **Standalone “Africa-wide Biological Control Programme Annual Report” series (PDF):** Not found online as a numbered open series. Coverage lives inside IITA ARs and EPMR text; CGSpace stub `10568/101872` has no file.
- **“IITA Plant Health Management Division Cotonou annual report” 1992–1995:** No open PDFs for those years. Citation evidence for a **1994 PHMD Annual Report** = EScaPP (Google Books). Open PHMD ARs begin ~1998/99.
- **USAID DEC (`pdf.usaid.gov`):** No ABCP/PHMD annual-report hit in this pass (cowpea volumes and unrelated biodiversity reports appeared instead).
- **World Bank:** Greathead & Waage 1983 *Opportunities for biological control…* (https://documents1.worldbank.org/curated/en/310251468739332721/pdf/multi_page.pdf) is background policy, **not** an IITA annual. Later Bank CGIAR impact papers cite African biocontrol but are not yearbooks.
- **biblio.iita.org Koha OPAC:** Search endpoints returned 404; some hashed document URLs still serve PDFs when known (e.g. 1996 AR).
- **ResearchGate / Cambridge / Elsevier:** Citations to Neuenschwander, Yaninek, Herren papers plentiful; many abstracts only / login walls—not used as primary annual-report substitutes here.
- **Firecrawl:** CLI not runnable here; relied on CGSpace API + search.
- **iitacbudm.github.io/CassavaHistoricalReports/**: Breeding-unit extract site indexing years 1971–2017; useful navigation aid, not a biocontrol report archive (page fetch timed out in this session).
